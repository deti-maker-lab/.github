# DETI Maker Lab

Welcome to the **DETI Maker Lab** GitHub organization!

## About the Project
The DETI Maker Lab is an open and free space designed for students, teachers, and staff to work on Electronics, Telecommunications, and Informatics projects. Currently, the laboratory operates using a Wiki-based platform for managing projects, equipment, requisitions, and documentation. 

Our project aims to improve this existing information system. The current requisition process is too complex and inconvenient to handle, and the Markdown-oriented data model makes equipment tracking harder than it should be. Our goal is to enhance the system without "reinventing the wheel" and preserving the installed base, while making it significantly more user-friendly for students and the lab technician.

## Key Objectives
* **Inventory Integration:** Integrate the Maker Lab system with Snipe-IT (https://snipeitapp.com) to replace the current wiki equipment management.
* **Simplified Workflow:** Enable a simple requisition flow (request -> approve -> assign -> return) that groups can easily use repeatedly throughout their project lifecycle.
* **Form-Based Input:** Replace Markdown project creation with form-based input to ensure consistent, database-ready data.
* **Mobile Apps:** Develop an iOS app for requisitions and improve the existing Android app. Requisitions made via mobile must work directly with the stock management tool.
* **Authentication:** Integrate with the university's "universal user" authentication.
* **Extra Services & Innovation:** Develop statistics pages and take on the advanced challenge of using blockchain technology to register requisitions and returns.
